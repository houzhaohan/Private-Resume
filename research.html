Research and Application of CLIP Pre-training Model Based on Hash Algorithm in Cross-modal Retrieval and Localization of Farmland Insect Images

Nanjing Agricultural University High Performance Computing Center

Graduation Design

Supervisor: Yihua Zhu (Associate Professor)

Aim to retrieve a large number of images of agricultural insects and locate them based on the results of image retrieval. The positioning results can be used to implement precision agriculture, such as targeted spraying and regulating the food chain. The system uses CLIP visual language model and deep hashing algorithm to extract multimodal features by fine-tuning the pre trained model, and designs a hierarchical hashing encoding mechanism to map high-dimensional features into compact binary hash codes. The project constructs a multimodal dataset containing 20000 agricultural crop images, implementing a GPU accelerated nearest neighbor search algorithm that supports text search and image multimodal retrieval. The system is developed using B/S architecture, and the backend is based on PyTorch to implement feature extraction and index management modules. The frontend provides a visual interactive interface to ensure retrieval speed while maximizing accuracy and recall. This system provides an efficient image retrieval solution for scenarios such as agricultural insect monitoring.

Key Achievements:
  • Developing information systems using B/S architecture

Analysis of Martingale Strategy in Quantitative Trading Market Based on BiLSTM-Attention Model

Hong Kong University of Science and Technology

Supervisor: Ke Yi (Professor)

This study proposes an improved Martingale Strategy based on the BiLSTM-Attention model, aimed at addressing the problem of insufficient adaptability of traditional Martingale Strategy under extreme market conditions. We have developed a predictive model that can capture the long-term temporal dependence of stock prices and designed a multidimensional dynamic management mechanism based on it. This mechanism integrates technical indicators such as RSI and MACD, volatility adjustment modules, and sliding window profit and loss strategies to achieve adaptive risk control.

Key Achievements:
  • Verified the feasibility of collaborative optimization between Deep Learning (BiLSTM-Attention) and traditional trading strategies (Martingale Strategy)
  • Provided a scalable technical framework for low-frequency quantitative trading, suitable for individual investors or lightweight financial product development
  • Published an EI Compendex paper [1-3]
